What is "uses" about?

Here you can find all sorts of information about the things that I use, or I find interesting and useful. You are always welcome to check this section out and see if there is something that you can apply in your setup/daily life.

Hardware (the one that is use)

  • I'm currently using a custom-build computer. Learning to build your own computer will save you money in the long run and probably help you make some money in certain cases.

  • I'm currently using a ThinkPad T520 as a mobile workstation, I also have a T61, but the battery on that one is dead. The T520 is currently running Debian 12 (xfce), which allows me to do most of the things that I need (edit documents, browsing the web, watch anime, etc).

  • My main keyboard is a Cherry G80-11900LUMEU-0 Wired Mini Keyboard and the mouse is a Logitech G502 X. I really enjoyed the old design of the keyboard and the mouse was recommended to me by a friend. Can't really say much more about them.

Software

  • I use both Windows 10 and Debian 12 as my operating systems. I mostly use Windows for gaming. I use Linux for pretty much everything else.

  • I recently switched to LibreWolf (a more privacy-focused firefox fork). I'm still using Startpage (a privacy focused search engine) as my main search engine.

  • Useful browser extensions to use: uBlock Origin to block ads and trackers, Bitwarden for password management, Leechblock to block all the unhealthy and distracting websites,Violentmonkey for running different scripts, LanguageTool free grammar checker that also offers a browser extension, SocialFocus gives you an undistracted social media experience, and Old Reddit Redirect does exactly what the title says.

  • Small list with scripts I have installed on Violentmonkey:AdsBypasser - Bypass Ads, Popups and count-down ads; Anti-Adblock Killer - Bypasses anti-adblock functionality in websites.

Applications

  • BleachBit - Clean Your System and Free Disk Space.
  • qBittorrent - aims to provide an open-source software alternative to µTorrent.
  • VLC - a free and open-source, portable, cross-platform media player software.
  • VSCodium - VSCodium is a community-driven, freely-licensed binary distribution of Microsoft’s editor VS Code.
  • Thunderbird - best open-source email client for linux and windows.
  • OBS - Free and open source software for video recording and live streaming.
  • Ninite - Install and Update All Your Programs at Once
  • Jdownloader2 - is a free, open-source download management tool. (I would also recommend a Real-Debrid subscription so you can download easier from premium file hosters)
